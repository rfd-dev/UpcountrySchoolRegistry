# UpcountrySchoolRegistryAPI
CASE Eleva - Demonstração técnica simulando um sistema para cadastro e consultas de escolas no interior do Pais.

## Objetivo
Permitir que um usuário cadastre e edite a informações sobre escolas e suas turmas e que seja capaz de consultar por escola, turma e aluno.

## Quickstart dos pacotes 3rd Party utilizados
* Swagger: https://docs.microsoft.com/en-us/aspnet/core/tutorials/web-api-help-pages-using-swagger?view=aspnetcore-3.1
* Serilog: https://github.com/serilog/serilog/wiki/Getting-Started
